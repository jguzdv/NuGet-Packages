# JGUZDV.Extensions.SAML2

This library provides extensions for working with SAML2 in .NET applications. It includes features such as:
- A metadata manager
- A certificate manager

You can use these extensions to simplify the process of integrating SAML2 authentication and authorization into your applications.
The metadata manager allows you to easily manage and retrieve SAML2 metadata, while the certificate manager helps you handle certificates for signing and encrypting SAML2 messages.

To see how it's used take a look at our [BundId-Connector](https://github.com/jguzdv/bundid-connector) project or [Passkey-Authentication-Provider](https://github.com/jguzdv/Passkey-Authentication).
